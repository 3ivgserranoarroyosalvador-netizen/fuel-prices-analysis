# 🌍 World vs Asia Fuel Prices — Análisis Comparativo

Análisis comparativo de precios de combustible a nivel mundial con enfoque en Asia,
usando MySQL, Python y Power BI.

## Stack tecnológico
- MySQL — almacenamiento y consultas SQL (15 queries)
- Python (Pandas, mysql-connector) — extracción de datos
- Power BI — visualización e insights

## Dashboard

## Insights principales
- Asia es **3.42% más barata** que el promedio mundial en gasolina
- **Europa** tiene la carga fiscal más alta: hasta 4 USD/litro en impuestos
- Los precios globales tuvieron su pico en **2022** post-pandemia
- Medio Oriente mantiene los precios más bajos por subsidios activos
